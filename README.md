# NestJS Application

A NestJS application with an Admin module.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Build

```bash
npm run build
```

## Modules

- **AdminModule**: A module for admin functionality
  - Controller: `/admin` endpoint
  - Service: AdminService with basic functionality

## API Endpoints

- `GET /admin` - Returns admin information
